DevOps Maturity User Guide

Use this guide to run the CLI, use the web interface, and keep your assessment workflow aligned with the current feature set.

Getting started

Choose your entry point

CLI workflow

Install and run an interactive maturity assessment from your terminal.

pip install devops-maturity
dm assess

Web workflow

Run the app locally and complete the guided form in your browser.

git clone https://github.com/devops-maturity/devops-maturity.git
cd devops-maturity
pip install nox
nox -s preview
Repository PyPI package

CLI guide

Run manual, config, and AI-powered assessments

Manual interview mode

Answer each weighted criterion interactively.

devops-maturity assess

YAML-driven mode

Use a versioned baseline in automation pipelines.

dm config --file devops-maturity.yml

Auto mode with AI

Use repository context and a configured provider.

OPENAI_API_KEY=... dm assess --auto --ai openai

Result output

Every mode returns score, level, category results, and badge URL for README usage.

Web guide

Complete and track assessments in the browser

  1. Open the Assessment tab and enter project details.
  2. Mark each practice as In place or Not yet.
  3. Submit to view maturity score, category breakdown, and improvement priorities.
  4. Use All Assessments to compare runs and edit historical entries.
